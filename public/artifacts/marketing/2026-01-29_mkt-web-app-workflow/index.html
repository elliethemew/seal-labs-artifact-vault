Deep Dive: The 3 Key Innovations

Don't Prompt Pixels. Prompt Logic.

The team didn't just ask for "a pretty website." They used Structural Description Prompts to define behavior.

// Example Prompt Structure

Context: "Create a landing page for a DeFi app. Style reference: Cyberpunk minimalism."

Logic: "The 'Join Waitlist' button must trigger a modal popup. The modal should have a z-50 index and a backdrop-blur effect."

Effect: "Use the 'Speed' background effect from the library with high intensity."

The Recursive Loop

Prompt → Review → Identify Layout Error → Prompt to Fix. (Don't try to get it perfect in shot #1).

Key Insight

Aura handles the HTML/CSS structure better than a human, but YOU must define the "State Management" (what opens what).

The "Zero-Cost" Database

JS wallet submission was complicated. The team pivoted to Google Apps Script to turn a simple Google Sheet into a database.

Frontend (Aura)
Google Script API
Google Sheet

The doPost(e) script captures the email/wallet and appends a row. No server maintenance required.

Why this is brilliant:

It decouples Marketing from Engineering. MKT can own the "Waitlist" data completely without asking Devs to query a SQL database.

Visual Debugging (The Screenshot Trick)

When the code looked "laggy" or wrong, they didn't write a bug report. They took a picture.

1. Screenshot

Capture the visual error.

2. Prompt

"Make the code match this image."

3. Fix

Claude generates the CSS fix.

Result: Erik & Quan fixed UI lag and tap effects without needing deep CSS knowledge.

For Designers / Creative

  • 1

    Create a "Component Map": Don't just prompt a whole page. Prompt section by section (Hero, Features, Footer) to keep Aura accurate.

  • 2

    Define States in Plain English: Write down "What happens when I click?" before you start generating.

For MKT Ops

  • 1

    Save the Backend Script: Copy the Google Apps Script code into Notion. Use it for every future waitlist.

  • 2

    Screenshot > Ticket: Stop writing long Jira tickets describing visual bugs. Just screenshot it and attach the Claude fix.
